# Building Time Series ML Models with sktime

Walking through forecasting with sktime using a sample sensor dataset.

## What's covered

- sktime data types and supported index formats
- Train-test splitting and forecasting horizons
- Preprocessing pipelines with imputation, detrending, and deseasonalization
- Fitting and comparing ExponentialSmoothing and ARIMA forecasters
- Time series cross-validation with `ExpandingWindowSplitter`

## Requirements

Python 3.10+ and:

```bash
pip install sktime pmdarima statsmodels
```
