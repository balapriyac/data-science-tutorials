# Time Series Data Cleaning with Python

Code for my freeCodeCamp article [How to Clean Time Series Data in Python](https://www.freecodecamp.org/news/how-to-clean-time-series-data-in-python/)

## Requirements

```
pandas
numpy
scipy
scikit-learn
statsmodels
```

## Install

```bash
pip install pandas numpy scipy scikit-learn statsmodels
```

## Topics Covered

Auditing, reindexing, missing value imputation, outlier detection and treatment, duplicate removal, frequency alignment, noise smoothing, and schema validation.
